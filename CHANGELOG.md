# Changelog

## April 28, 2023 - Forked
Forked from [here](https://www.romhacking.net/community/5598/)

## Dec. 17, 2022 - v3.3
Major bug fixes. DE type enable codes should now work correctly and fixed a bug where a bad gameshark code, comment, or empty line in the gameshark codes list would cause all the subsequent codes to not get injected.

## Dec. 16, 2022 - v3.2
Fixed compatibility issue with Goldeneye involving not all exception handler hooks being used. I put in checks for each hook.  Might also fix other games.

## Nov. 29, 2022 - v3.1
Added compatibility with Shadows of the Empire (U) v1.0, v1.1, & v1.2.

## Nov. 27, 2022 - v3.0
Added compatibility for Mario Tennis (USA) and Perfect Dark (USA) with hardcoded solutions.  Future versions will hopefully increase compatibility further.  Modified how the Gameshark Code Injector runs at boot to for increased compatibility and in-game performance.  Now you can add comments and blank lines to your gameshark codes and the injector will ignore them instead of crashing.

## Dec. 7, 2021 - v2.0
Now supports almost every game and many more code types including F0, F1, A0, A1, FF, EE, and DE.

## Oct. 29, 2021
Mario Golf version created.

## Jan 6, 2020 - v1.0
Initial Release (MK64 version)
